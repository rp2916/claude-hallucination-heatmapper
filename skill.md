# Instructions

1. When triggered, do not immediately answer the user's prompt yourself. 
2. Instead, ask the user to provide an Anthropic API key if they haven't already (since the sandbox script needs to make background API calls).
3. Once you have the key and the prompt, execute `heatmapper.py` in the code execution environment. Pass the user's original prompt and the API key as arguments.
4. Take the raw HTML output generated by `heatmapper.py` and display it directly to the user so the HTML color tags render properly (showing green, yellow, and red text formatting).
